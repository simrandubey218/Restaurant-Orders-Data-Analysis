This project showcases advanced SQL techniques used to analyze and derive critical **Key Performance Metrics (KPMs)** from a transactional database of restaurant orders for an e-commerce food delivery platform.
The analysis was designed to gauge the performance of a new vertical launched in Dubai on January 1st, focusing on customer acquisition, retention, and channel performance.

Case - Noon launched food items in Dubai on 1st of January and the line manager has asked to share key performance meterics to gauge the performance of the verticals. All the data of orders is being stored in the below table which is used to write queries and find insights.
